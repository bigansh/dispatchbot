Features

Create a DM request

With the !dis dm @user command, create a DM request. The channel is then created if the requested user approves your request.

Delete a channel

With the !dis del command, delete the DM channel. Once all the users accept the delete request, the channel gets deleted.

Access help

With the !dis help command, access the help menu whenever needed. This menu contains all the command the bot understands.

Create Group DMs

With the !dis gdm @user, @user command, you can request for a group DM. If all the users mentioned accept the request, a channel is created only visible to the mentioned members.

Frequently Asked Questions

Is it free?

Yes, using the Dispatch bot is completely free. Even if we introduce some more features that we think should have a paywall, the core features will always be available for free.

Who all can see that chat?

Apart from the requested entities and the admins, no-one else can see the chat. We would like to make it not visible to admins as well, but Discord's current limitations don't allow us to do so.

Can I delete a channel?

Deleting a channel is just as easy as creating one. You just need to create a request for the same using the !dis del command. Once all or more than have the members agree, the channel is deleted.
